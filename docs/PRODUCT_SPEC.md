# 数据终端 - 产品规划文档

## 项目目标

建立一个统一的数据获取系统，从多个数据源拉取选定标的的信息，每日自动更新K线数据，并提供给其他系统（回测系统、基金管理系统）使用。

---

## 数据分类

### 一、具体标的（Individual Assets）

#### 1. 基金
- 价格数据
- 净值（NAV）
- 费率信息

#### 2. 股票
- **美股**
  - 价格数据（日K、分钟K）
  - 基本面（PE、PB、EPS、营收、利润）
- **A股**
  - 价格数据
  - 基本面（PE、PB、ROE）

#### 3. 加密货币
- 价格数据（多时间维度）
- 链上数据（可选）
  - 交易所流入/流出
  - 大额转账
  - 持仓分布

#### 4. 大宗商品
- 黄金
- 白银
- 原油
- 其他商品期货

---

### 二、市场整体情况（Market Overview）

#### 情绪指标
| 指标 | 说明 | 分档 |
|-----|------|------|
| BTC 恐慌贪婪指数 | 加密货币市场情绪 | 极度恐惧、恐惧、中性、贪婪、极度贪婪 |
| VIX | 标普500波动率指数（恐惧指数） | 低、中、高、极高 |
| 黄金波动率指数 | 类似GVIX | 低、中、高、极高 |

#### 估值指标（200周均线分档法）
| 资产 | 指标 | 分档定义 |
|-----|------|---------|
| BTC | 200周均线偏离度 | 极度便宜(<-50%)、便宜(-50%~-20%)、合理(-20%~+20%)、昂贵(>+20%) |
| SP500 | 200周均线偏离度 | 极度便宜(<-30%)、便宜(-30%~-10%)、合理(-10%~+10%)、昂贵(>+10%) |
| 黄金 | 200周均线偏离度 | 同上 |

#### 市盈率估值
| 市场 | 指标 | 分档 |
|-----|------|------|
| 标普500 | PE Ratio | 便宜(<15)、合理(15-25)、昂贵(>25) |
| 沪深300 | PE Ratio | 便宜(<12)、合理(12-18)、昂贵(>18) |

#### 其他市场情绪指标
- 融资融券余额（A股）
- 融资买入额占比
- 北向资金流向
- 新开户数

## 系统架构设想

```
数据终端
├── 数据采集层（多个 fetcher）
├── 数据存储层（SQLite/PostgreSQL）
├── 数据处理层（指标计算、分档评估）
├── API 服务层（FastAPI 提供数据接口）
└── 定时任务层（每日自动更新）
```

---

## MVP 阶段目标

1. 支持 4 类基础标的：BTC、GOLD、SPY、HS300
2. 日K数据自动更新
3. 基础情绪指标（BTC恐慌贪婪、VIX）
4. 200周均线估值分档
5. REST API 供回测系统调用
